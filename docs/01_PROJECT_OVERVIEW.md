# Project Overview

## 프로젝트명
AI Sentiment Insight

## 프로젝트 목표
사용자가 텍스트를 입력하면 OpenAI 기반 감성 분석 결과를 제공하는 웹 서비스 구축

## 핵심 기능
- 텍스트 입력
- 감성 분석 API 호출
- 긍정/부정/중립 결과 표시
- 신뢰도 표시
- 분석 이유 제공
- 결과 모달 출력

## 기술 스택
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- AI: OpenAI API
- Database: Supabase
- Deploy: Vercel

## MVP 범위
- 단일 페이지 웹앱
- 감성 분석 기능
- 결과 저장

## 제외 범위
- 로그인
- 다국어
- 통계 기능
- 관리자 기능

## DONE 조건
- 메인 화면 구현
- 감성 분석 동작
- 결과 모달 출력
